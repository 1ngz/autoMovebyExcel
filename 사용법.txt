Auto Creation 매크로 사용법

필요 파일은 Auto Creation v0.0.4(이하 메인 프로그램)과 명령어 테이블(엑셀)입니다.

메인 프로그램에서 Ctrl+Q 버튼을 통해 얻은 좌표를
엑셀 파일로 도식화 합니다.

2열부터 순차적으로 채워야하며, A열은 명령어, B,C열은 좌표를 입력하시면 됩니다.

<명령어 종류>
Left : 좌클릭(DLeft : 더블클릭) / Right : 우클릭 / Key : 키보드 입력


키보드는 B열에 값을 입력합니다.

<키보드 명령어 종류>
추가로, Copy, Paste, Select, Enter 로 넣으면 각각 Ctrl+c / Ctrl+v / Ctrl+a / Enter 키 입력이 됩니다.


- 시작: F3
- 강제종료: F4
